# 🎮 AirClash

AirClash - это динамичная 2D игра в стиле аэрохоккея с быстрым геймплеем, соревновательной системой и множеством режимов.



## 🚀 Особенности

- ⚡ Быстрый и отзывчивый геймплей
- 🤖 Боты с уровнями сложности:
  - Очень лёгкий
  - Лёгкий
  - Средний
  - Сложный
  - Экстрим
- 🧠 Умный ИИ с “человечным” поведением
- 🏆 Эло-система (только для онлайн-матчей) (в разработке)
- 🌍 Глобальный и региональный рейтинг (в разработке)
- 🎮 Несколько режимов: (в разработке)
  - 1v1 (классика)
  - 2v2 (напарники)
  - 3v3 (команда)
  - 5v5 (ежемесячные ивенты)
- 🌐 Онлайн мультиплеер (в разработке)
- 📱 Поддержка мобильных устройств (Android)



## 🎯 Цель проекта

Создать простую в освоении, но сложную в мастерстве игру, которая может стать соревновательной и даже киберспортивной.



## 🛠️ Технологии
