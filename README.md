# 🎮 AirClash

AirClash - это динамичная 2D игра в стиле аэрохоккея с быстрым геймплеем, соревновательной системой и множеством режимов.



## 🚀 Особенности (в разработке)

- ⚡ Быстрый и отзывчивый геймплей
- 🤖 Боты с уровнями сложности:
  - Лёгкий
  - Средний
  - Сложный
  - Экстрим
  - Кошмар
- 🧠 Умный ИИ с “человечным” поведением
- 🏆 Эло-система (только для онлайн-матчей)
- 🌍 Глобальный и региональный рейтинг
- 🎮 Несколько режимов:
  - 1v1 (классика)
  - 2v2 (напарники)
  - 3v3 (команда)
  - 5v5 (ежемесячные ивенты)
- 🌐 Онлайн мультиплеер
- 📱 Поддержка мобильных устройств (Android / iOS)



## 🎯 Цель проекта

Создать простую в освоении, но сложную в мастерстве игру, которая может стать соревновательной и даже киберспортивной.



## 🛠️ Технологии

- Unity ❤️ (2D)
- C#


## 📦 Установка

1. Клонируйте репозиторий:
``` git clone https://github.com/Zebraar/AirClash.git ```
2. Откройте проект в Unity Hub
3. Запустите сцену через Unity Editor
   Скоро сделаю Android порт в .apk файл
